Criar um repositório github
clonar o prjeto no PC -> git clone "nome_do_repositório_criado"
configuração usuario -> git config user.name "seu_nome"
configuração email -> git config user.email "nome@email.com"
vai repetir varias vezes.
Verificar ramo atual -> git branch
criando um novo ramo e entrar -> git checkout -b "nomebranch"
Adicionar arquinos de codigo
adicionar alterações para comitar -> git add .
Commitar codigo -> git commit -m"feat: mensagem"
enviar codigo para o git -> git push origin "nomebranch"
retornar para a main -> git checkout main
atualizar branch main -> git pull origin main
